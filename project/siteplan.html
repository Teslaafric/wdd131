Site Name

BigBird Technologies Solutions

Site Purpose

Represents a professional computer repair and IT support service center. This page will enable the institution to communicate to customers and other visitors about computer repair services offered. View available repair services, learn about pricing, contact the company, schedule repair appointments.

Scenarios

  • My laptop is running very slow. Can this technician fix it?
  • How much does it cost to repair a broken computer screen?
  • Where can I contact the technician to book an appointment?

Color Schema

  • Dark Blue #0A192F Header & Footer background.
  • Bright Blue #1E90FF Buttons & Navigation hover.
  • Light Gray #F4F4F4 Page background.
  • White #FFFFFF Card background.

Typography

  • Primary Font: Roboto - Body text
  • Secondary Font: Montserrat - Headings
